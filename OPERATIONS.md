# 운영 가이드 — 0원으로 시작하는 24/7 자동 수익 시스템

이 시스템이 실제로 돈을 벌기 시작하려면 **세팅 1회 + 첫 한 달 마케팅**이 필요합니다.
완전 자동화는 1개월 차부터 가능합니다.

## 1주차: 인프라 세팅 (총 1~2시간)

| 작업 | 소요 | 비용 |
|---|---|---|
| 쿠팡파트너스 가입 → 파트너 ID 발급 | 10분 | 0 |
| GitHub 레포 fork/clone → Secrets 등록 (`ANTHROPIC_API_KEY`, `COUPANG_PARTNERS_ID`) | 5분 | 0 |
| GitHub Pages 활성화 (Settings → Pages → Source: GitHub Actions) | 2분 | 0 |
| 첫 발행 수동 트리거 → 글 품질 확인 | 10분 | API 사용량 약 ₩30 |
| Vercel 가입 → `saas/` 디렉토리 import → 환경변수 입력 → 배포 | 15분 | 0 |
| Google Search Console / 네이버 서치어드바이저 사이트 등록 + sitemap 제출 | 30분 | 0 |
| (선택) 도메인 구매 + 연결 | 10분 | 연 ~₩15,000 |

## 2~4주차: 트래픽 시드 (매일 10~20분)

자동 발행되는 글이 검색 엔진에 색인되기까지 보통 1~3주.
이 기간엔 **수동 보조**가 필요합니다.

- 매일 새벽 봇이 글 1개 자동 발행 → 아침에 1회 훑어보고 제목·메타만 다듬기 (5분)
- 트위터/스레드/네이버 카페에 글 링크 1회씩 공유 (10분)
- 1주에 한 번, `content-bot/src/trends.py` 의 `FALLBACK_TOPICS` 를 본인 niche 에 맞게 교체 (계절성/시즌이슈)

## 2개월차~: 자동 운영

- 글 30개 이상 누적 + Search Console 에 키워드 노출이 잡히기 시작
- 쿠팡파트너스 수익 트래킹 (파트너스 대시보드)
- SaaS 도구별 사용량 모니터링 (Vercel Analytics 무료)
- 수익성 좋은 키워드 위주로 `FALLBACK_TOPICS` 가중치 조정

## 확장 옵션 (수익이 손익분기 넘으면)

1. **AdSense 추가** (블로그 트래픽 누적 후 신청, 보통 1000 PV/일부터 의미)
2. **SaaS Pro 마케팅** — 첫 100명 평생 50% 할인 같은 후크
3. **신규 도구 추가** — `saas/lib/prompts.ts` 에 1줄 추가하면 즉시 출시
4. **콘텐츠 채널 확장** — YouTube Shorts 자동 생성 (`pyttsx3` + `moviepy` 로 확장 가능)

## 솔직한 기대치

- 1개월차: 수익 ₩0~₩5,000 (글이 인덱싱 시작)
- 3개월차: ₩50,000~₩300,000 (트래픽 안정화 + SaaS 첫 유료 사용자 가능)
